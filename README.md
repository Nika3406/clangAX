# ClangAX: C-Accel Compiler
## Installation
1. Download Zip and unzip in preferred folder
2. Go to Terminal and go to the directory of that folder
3. To compile any .cax file type this in Terminal
```
user$ ./cmake-build-debug/clangax SampleCode.cax -o programName 
```
4. Once compilation is done type ./programName to see how it runs
