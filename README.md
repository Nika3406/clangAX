# ClangAX: C-Accel Compiler
## Installation
1. Download Zip and unzip in preferred folder
2. Make sure to have downloaded LLVM, if you're on Mac just brew install llvm
3. Go to Terminal and go to the directory of that folder
4. To compile any .cax file type this in Terminal
```
user$ ./cmake-build-debug/clangax SampleCode.cax -o programName 
```
5. Once compilation is done type ./programName to see how it runs
